# 🐾PawHaven

## 🔗 [Live Demo](https://ikerom3.github.io/Project-GalaxyTeam/)

## 📌About the project

PawHaven - is a web application that helps people find dogs from shelters and
adopt them into their families. The platform brings together shelters and future
owners in one convenient service.

The project was created as a full-stack application to demonstrate modern web
development.

## 🛠Teach Stack

- HTML5 & CSS3 (SASS/SCSS)
- JavaScript (ES6+)

- BEM Methodology - For organized, modular, and scalable styling.
- Vite - Modern build tool used for optimized development performance.
- Featured Libraries - Includes Swiper.js for sliders, SweetAlert2 for
  notifications, and basicLightbox.
